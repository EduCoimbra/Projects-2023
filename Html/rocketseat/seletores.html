Combinators

It searchs all <h2> elements inside <article> and <section>

section article h2 { }

Child Combinator

It just selects elements that are direct child

nav > ul > li {
  color: red;

Adjacent Sibling Combinator

It just selects adjacent siblings (at the right) of the element

.button + .button { }

General Sibling Combinator

It selects all adjacent siblings (at the right) of the element, even if there is another element between them

.button ~ .button { }
