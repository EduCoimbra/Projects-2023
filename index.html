HTML

Web Semântica

<header>

É utilizado para representar o cabeçalho de um documento ou seção declarado no HTML. Geralmente é no Header principal da página que colocamos um menu de navegação.

<nav>

É a tag que usamos para inserir um menu de navegação.

<main>

Essa tag especifica o conteúdo principal e, consequentemente, de maior relevância dentro da página. Para ser considerada bem construída, uma página deve apresentar apenas um conteúdo principal.

<aside>

Relacionado levemente com o conteúdo main:

  • - Explicações
  • - Glossários
  • - Links extras
  • - Biografia do autor
  • - Informações do perfil

<article>

Utilizamos o elemento quando precisamos declarar um conteúdo que não precisa de outro para fazer sentido em um documento HTML. É recomendado identificar cada um com um título. Veja algumas possíves aplicações:

  • - receitas
  • - posts
  • - conteúdos de fotografia
  • - é uma composição independente

<section>

Pode ser qualquer seção de uma página, como uma seção de Contato, Fale conosco ou uma faixa que está linkada a um menu.

<div>

Não é considerada uma tag semântica e pode ser usada de forma independente como para um carrinho de compras, por exemplo.

<figure>

É uma marcação de uso específico para a inserção de uma figura/imagem. Para incluir a descrição dessa figura, podemos utilizar o elemento <figcaption>.

<footer>

Esse elemento representa o rodapé de um documento. Normalmente é utilizado para descrever informações como:

  • - a data de criação do conteúdo
  • - informações do autor
  • - copyright
  • - contato
  • - sitemap
  • - voltar ao topo

Web Semântica no nível do texto

<a>

A principal função do elemento é descrever um link, conectando os diversos documentos de um site e permitindo a navegação por esse conteúdo. Normalmente esses documentos estão relacionados por compartilharem um assunto em comum.

<em>

O elemento é utilizado quando desejamos enfatizar um trecho ou palavra no texto, indicando que ela contribui de forma mais relevante para o sentido/compreensão do conteúdo.

<strong>

O elemento também é utilizado para destacar uma parte do texto. Sua principal diferença em relação ao elemento <em> é que <em> pode alterar o propósito de uma frase, como vimos anteriormente.

<cite> e <q>

O elemento <cite> é utilizado para declarar que naquele trecho há uma citação, isto é, um trecho de texto que não foi escrito pelo autor do conteúdo. Normalmente o utilizamos em conjunto com o elemento <q>, responsável por apresentar o conteúdo retirado de outra fonte.

<time>

O elemento é utilizado para representar datas. Assim, caso seja necessário informar a data em que um conteúdo foi escrito, podemos declarar a tag <time> e acrescentar a ela o atributo datetime para escrever a data de forma padronizada.
